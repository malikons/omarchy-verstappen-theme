# Verstappen Theme for Omarchy

A dark, aggressive theme inspired by **Max Verstappen** and **Red Bull Racing** — four-time Formula 1 World Champion.

Deep navy darkness like a night race. Fiery red like the livery. Dutch orange like the fans in the grandstand. Championship gold for the one who takes the win.

![Verstappen Theme Preview](theme.png)

## Color Palette

| Role | Color | Hex |
|------|-------|-----|
| Background | Deep Navy | `#0a0e1a` |
| Accent | Red Bull Red | `#e8002d` |
| Highlight | Dutch Orange | `#ff6600` |
| Special | Championship Gold | `#ffd700` |
| Foreground | Cool White | `#e8eaf0` |

## Installation

```bash
omarchy-theme-install https://github.com/malikons/omarchy-verstappen-theme
```

Or manually:
```bash
cp -r omarchy-verstappen-theme ~/.config/omarchy/themes/verstappen
```

Then select **verstappen** via `Super + Ctrl + Shift + Space`.
